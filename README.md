Este projeto apresenta uma calculadora de juros compostos desenvolvida em HTML, CSS e JavaScript, com exibição dinâmica em tabela para acompanhamento diário do crescimento do valor investido.
O usuário pode escolher entre duas opções de investimento:

SOX36: juros de 4,5% ao dia e comissão de 2,5%.
TSVUSDT: juros de 6,0% ao dia e comissão de 4,0%.

Ao informar o valor inicial e o número de dias, a aplicação calcula automaticamente a evolução do capital líquido e bruto, exibindo os resultados dia a dia.
Projeto criado para estudos de finanças e aplicações web interativas.
